Kliko's modloader

An open-source modloader for Roblox

Kliko's modloader is an open-source modloader for Roblox. It can be used to conveniently launch a modded copy of Roblox. Python 3.12 and a PC running Windows is required to run this program.

Features

Mod Profiles

Quickly switch between pre-configured mod profiles through the modloader menu.

FastFlags support

Set up multiple FastFlag profiles. For now, FastFlag profiles will have to be set up manually by editing the config file.

Rich Presence

Show your friends on Discord that you are using Kliko's modloader.

Installation Guide

Follow these instruction to install Kliko's modloader

Step one

Download and install the latest release!

Step two

Run 'Modloader Menu' to change settings or apply mod profiles, run 'Launcher' to launch modded Roblox.

Step three (optional)

During the installation process you can choose to add shortcuts to your desktop and start menu.

Goals & Inspiration

Goals

My goals for this project are:

  • Introducing more people to Roblox mods

  • Getting more experience with Python

  • Learning more about managing discord servers

  • Making a website

Inspiration

This project was inspired by Bloxstrap, an open-source, alternative bootstrapper for Roblox. Bloxstrap is what introduced me to the Roblox modding community and where I learned many different ways to improve my mods. Not being able to quickly switch between custom mod profiles and set mod priorities is what led to me creating my own modloader.
