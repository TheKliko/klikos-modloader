Kliko's modloader

An open-source modloader for Roblox.

Kliko's modloader is an open-source modloader for Roblox. It can be used to conveniently launch a modded copy of Roblox. Python 3.12 and a PC running Windows is required to run this program.

Installation Guide

Follow these instructions to install Kliko's modloader.

step 1


Download and extract the latest release.

  • The latest version can be found at:

    • The Download page of this website

    • The official GitHub page

step 2


Open the (extracted) .zip file and move the folder inside it to where you wish to store this program.

  • Example: ("%localappdata%\Kliko's modloader")

step 3


Search for the 'Launch Options' folder and choose which options you want to add to your desktop or pin to the start menu.

  • Run the Setup shortcut to select a mod profile, reinstall Roblox, change settings, and more.

  • Run the Launcher shortcut to launch roblox.

  • Run the Info shortcut to get info on your applied profiles and settings.

Goals & Inspiration

Goals


My goals for this project are:

  • Introducing more people to Roblox mods

  • Getting more experience with Python

  • Getting experience in managing discord servers

  • Learning how to make a website

Inspiration


This project was inspired by Bloxstrap, an open-source, alternative bootstrapper for Roblox. Bloxstrap is what introduced me to the Roblox modding community and where I learned many different ways to improve my mods. Not being able to switch between custom mod profiles and mod priorities is what led to me creating my own modloader.
