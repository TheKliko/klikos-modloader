Modloader Menu

The modloader menu is the place where you manage your mods and configure your settings.

Mod Profiles

Mod profiles can be configured directly through the modloader menu. After selecting a profile, you can view all mods, add/remove mods, change mod priorities and rename or delete the mod profile.

Creating Mod Profiles

To create a mod profile, open the menu and navigate to Configure Mod Profiles > Create New Profile. Now, enter the name of your profile. If a profile with the same name does not exist, a new one will be created.

Adding Mods

To add a mod from to Profile, navigate to Configure Mod Profiles and select a profile. Choose 'Add Mods' and you will see a list of every mod found in your mods folder that is not already in the selected profile. From here, simple enter the number corresponding to your mod and it will automatically be added with the first available priority number (starting with 1). Removing mods works similarly, except you choose the 'Remove Mods' option to see a list of every mod in the selected profile.

Mod Priorities

Mod priorities are what makes sure your mods get applied in the right order.

  • Mods with a higher priority number will get applied first
  • Mods with a lower priority number will get applied later and may overwrite mods with higher priority numbers
Mod priorities can be set to any integer number, including negative numbers.

FastFlag Profiles

FastFlags are settings that can be used to quickly change the way that the client functions, mainly intended to be used by Roblox engineers. Due to a majority of people abusing this feature to gain unfair advantages in games, Roblox plans to disable the ability to set FastFlags locally. When this change happens, the ability to set up FastFlag profiles in Kliko's modloader will be removed as well.

Creating FastFlag Profiles

Creating FastFlag profiles can be done directly through the modloader menu. To do so open the menu and navigate to FastFlags > Configure FastFlag Profiles > Create New Profile. Enter the name you want to give to this profile and press ENTER.

Adding FastFlags

To add FastFlags to your profile, go to FastFlags > Configure FastFlag Profiles and select the profile that you want to add your fflags to. Now, choose 'Add FastFlags'. First you will be asked to enter the name of your fastflags, after that you can enter it's value.

Importing FastFlags

To import FastFlags, select your FastFlag profile and choose the option 'Import FastFlags'. You will now be prompted to open a .json file

Exporting FastFlags

To Export FastFlags, select your FastFlag profile and choose the option 'Export FastFlags'. Your FastFlags will be exported to a .json file

Mods

Mods are a way to personalize your Roblox experience. They are quite limited in what you can change, but — with the right set of mods — you can make Roblox look awesome. The most common mods that you will find are mods that affect UI elements and character sounds, as showcased below.

Downloading Mods

You can find Roblox mods on sites like GameBanana, in our Discord server or in servers for other programs that share the same functionality (example: Bloxstrap). The mods you've downloaded should be in the form of a .zip file. Follow these steps to install them:

  1. Download and extract the .zip file
  2. Open File Explorer and go to "%localappdata\Kliko's modloader\Mods%"
  3. Drag and drop the (extracted) .zip file into your mods folder

Mod Formatting

Mods should be added to your mods folder with the right file structure. To verify that your downloaded mods will get applied correctly, please make sure that they follow the format shown below.

    • Mod
      • content
      • ExtraContent
      • . . .
    • Mod
    • . . .
example: You have a mod that changes the death sound. It should be formatted like this:
"mod_name\content\sounds\ouch.ogg"

Common Errors

Program crashes immediately (no error message)

Make sure that Python is (correctly) installed before trying to launch Kliko's modloader. If Python is already installed, and this issue still happens, please follow these steps:

  • Open file explorer and go to "%localappdata%\Kliko's modloader"
  • Right click anywhere, select open in terminal and type the following command: python "Program Files\main.py"
  • It will now show the error message without instantly closing the terminal.
"Python was not found"
This means that windows does not recognize the python command. Please reinstall python and make sure to check the box that says 'Add python.exe to PATH', as shown in the image below.

PlatformError

As of writing this, Kliko's modloader is only supported on Windows platforms, with no plans of adding support for other systems. Because of this, users attempting to launch this program on an unsupported OS will receive this error message.

ValidationError

This error occurs when (one of) the required files could not be found. Simply reinstalling Kliko's modloader should fix this issue.

NotImplementedError

You shouldn't encounter this error. If you do, please report this in our Discord server

Unexpected errors

Unexpected errors are any errors not listed above. To get more information on why you received this error message, please read the latest log file in "%localappdata%\Kliko's modloader\Logs". If you need any assistance, you are welcome to join our support server and let us know.
