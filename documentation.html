Adding Mods

  1. Go to "%localappdata%\Kliko's modloader\Mod Profiles".
    • This is where your mod profiles are stored.
    • To create a new mod profile, simply create a new folder (CTRL+SHIFT+N) and rename it.
  2. Open the folder of the mod profile that you want to add your mod to.
    • It is important that you put the mod inside a mod profile.
    • Putting the mod in the mod profile directory without adding it to a mod profile will result in the mod not getting applied properly.
  3. Add the mod to your mod profile.
  4. Run 'Modloader Menu' and select your mod profile.
  5. Go back and choose 'Save & Exit'.

Mod Formatting

Make sure your mods follow the correct format.

Mods should be made with the following file structure:

  • "%localappdata%\Kliko's modloader\Mod Profiles"
    • Mod Profile
      • Mod
        • content
        • ExtraContent
        • . . .
      • Mod
      • . . .
    • Mod Profile
    • . . .

example: You have a mod that changes the death sound

It should be formatted like this: "Mod\content\sounds\ouch.ogg"

Mod Priorities

Mod priorities allow your mods to get applied in a specific order.

  • The mod with the lowest number (eg: 1) gets applied last.
  • The mod with the highest number (eg: 10) gets applied first.
  • Mods that do not have a set priority get applied before priority mods in alphabetical order.

Setting up Mod Priorities

  1. Go to "%localappdata%\Kliko's modloader\Program Files\config" and open 'mod_profiles.json'.
  2. Enter your priority mods with the following format:
    { { "Mod Profile": { "Mod Priority" : "Mod", "Mod Priority" : "Mod" }, "Mod Profile": { "Mod Priority" : "Mod" } } }
    • Replace 'Mod Profile' with the name of your mod profile.
    • Replace 'Mod' with the name of your mod.
    • Replace 'Mod Priority' with the number representing your mod's priority.
    • 'Mod Profile' and 'Mod' are case sensitive.
    • 'Mod Priority' needs to be the string of an integer. ("1", "2", etc.)
      • negative numbers can be used too.
    • A comma (,) is required between individual elements.
      • in this case: use a comma inbetween each mod and mod profile.

FastFlags

Adding FastFlags

  1. Go to "%localappdata%\Kliko's modloader\Program Files\config" and open 'fastflag_profiles.json'.
    • This is where your FastFlag profiles are stored.
  2. Edit your FastFlag profile, make sure to use the following format:
    { { "FastFlag Profile": { "FastFlag Name": "FastFlag Value", "FastFlag Name": "FastFlag Value" }, "FastFlag Profile": { "FastFlag Name": "FastFlag Value" } } }
    • Replace 'FastFlag Profile' with the name of your FastFlag profile.
      • case sensitive.
    • Replace 'FastFlag Name' with the name of your FastFlag.
    • Replace 'FastFlag Value' with the value of your FastFlag.
    • A comma (,) is required between individual elements.
      • in this case: use a comma inbetween each FastFlag and FastFlag profile.
  3. Run 'Modloader Menu' and select your FastFlag profile.
  4. Go back and choose 'Save & Exit'.
