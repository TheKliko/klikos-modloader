Modloader menu

The modloader menu is where you can choose and/or configure your mod profiles and FastFlag profiles. You can also use it adjust settings and access helpful information.

Setting up Mod Profiles

Mod profiles can be configured directly through the modloader menu. All you have to do is navigate to Mods > Configure Mod Profiles. From here, you can create and configure your profiles. After selecting a profile, you can view all mods, add/remove mods, change mod priorities and rename or delete the mod profile.


Creating Mod Profiles

To create a mod profile, open the menu and navigate to Mods > Configure Mod Profiles > Create New Profile. Now, enter the name of your profile, if a profile with the same name does not exist, a new one will be created.


Adding / Removing Mods

Adding mods

To add a mod from to Profile, navigate to Mods > Configure Mod Profiles and select a profile, now choose 'Add Mods' and you will see a list of every mod found in your mods folder that is not already in the selected profile. From here, simple enter the number corresponding to your mod and it will automatically be added with the first available priority number (starting with 1).

Removing mods

To remove a mod from your Profile, navigate to Mods > Configure Mod Profiles and select a profile, now choose 'Remove Mods' and you will see a list of every mod in the selected profile. From here, simple enter the number corresponding to your mod and it will be removed from your profile.


Mod Priorities

Mod priorities are what makes sure your mods get applied in the right order.

Mods get applied in the following order: high â†’ low

  • Mods with a higher priority number get applied first
  • Mods with a low priority number get applied last

Mod priorities can be set to any number, including negative numbers.

Setting up FastFlag Profiles

FastFlags are settings that can be used to quickly change the way that the client functions, mainly intended to be used by Roblox engineers.
Due to a majority of people abusing this feature to gain unfair advantages in games, Roblox plans to disable the ability to set FastFlags locally.
When this change happens, the ability to set up FastFlag profiles in Kliko's modloader will be removed as well.


Creating FastFlag Profiles

Creating FastFlag profiles can be done directly through the modloader menu. To do so open the menu and navigate to FastFlags > Configure FastFlag Profiles > Create New Profile.
Enter the name you want to give to this profile and press ENTER.


Adding FastFlags

To add FastFlags to your profile, go to FastFlags > Configure FastFlag Profiles and select the profile that you want to add your fflags to.
Now, choose Add FastFlags. First you will be asked to enter the name of your fastflags, after that you can enter it's value.
Disclaimer: Do not blindly add flags if you do not know what they change.

Changing Settings

Settings can be changed through the modloader menu by navigating to the Settings section.

Currently, the following settings are available:

  • max_log_files: The maximum number of log files that will be stored in the logs directory
  • launch_roblox_after_setup: Launches Roblox after exiting the modloader menu
  • auto_install_roblox_updates: Skips the confirmation prompt when installing Roblox updates
  • roblox_reinstall_after_changes: Reinstalls Roblox after selecting a different mod profile (recommended)
  • discord_rpc: This setting decides whether discord rich presence should be used
  • default_launch_arguments: Sets the default launch arguments for this program

Downloading Mods

You can find Roblox mods on sites like GameBanana, in our Discord server or in servers for other programs that share the same functionality (example: Bloxstrap).
The mods you've downloaded should be in the form of a .zip file. Follow these instructions to install them:

  1. Download and extract the .zip file
  2. Open 'File Explorer' and go to "%localappdata%\Kliko's modloader\Mods"
  3. Drag and drop the (extracted) .zip file into your mods folder

To verify that your downloaded mods will get applied correctly, please make sure that they follow the right format as stated below under 'Mod Formatting'.

Mod Formatting

Mods should be added to your mods folder with the following file structure:

  • ("%localappdata%\Kliko's modloader\Mods")
    • Mod
      • content
      • ExtraContent
      • . . .
    • Mod
    • . . .
example: You have a mod that changes the death sounds
It should be formatted like this: "Mod\content\sounds\ouch.ogg"

Troubleshooting

SystemError

As of writing this, Kliko's modloader is only supported on Windows platforms, with no plans of adding support for macOS.
Because of this, users attempting to launch this program on an unsupported OS will receive this error message.

SettingsNotFoundError / VersionInfoNotFoundError

This error occurs when required config files could not be found.
Reinstalling the program should fix this issue.

UpdateAvailableError

This error message is shown when a newer version of Kliko's modloader is available, the url to the latest release will automatically be openede in your default browser.
If, for any reason, you do not want to update, there are two ways to bypass this error.

  • Temporary solution

    Open 'File Explorer' and go to "%localappdata%\Kliko's modloader\Program Files\config", open the file named version.json.
    Change the value of "version" to that of the latest version, this will prevent the exception from happening until another update is released.

  • Permanent solution

    Open 'File Explorer' and go to "%localappdata%\Kliko's modloader\Program Files\modules\utils", open the file named startup.py.
    Comment out the following lines of code:
    • 47   try:
    • 48       updater.check_for_update()
    • 49
    • 50   except LatestVersionNotFoundError as e:
    • 51       logging.warning('Failed to get the latest version')

RobloxOutdatedError

This error occurs when the user refuses to update to a newer Roblox version

LaunchArgumentError

This happens when the user tries to launch Kliko's modloader with invalid launch arguments.

NotImplementedError

If you receive this error message, it means that I have accidentally released an incomplete version of Kliko's modloader, please join the discord server to report this issue.

Unexpected errors

Unexpected errors are any errors not listed above.
To get more information on why you received this error message, please read the latest log file in "%localappdata%\Kliko's modloader\Logs".
If you need any assistance, you are welcome to join our support server and let us know.
