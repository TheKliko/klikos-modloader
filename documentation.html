Adding Mods

Quick Guide

  1. Go to "%localappdata%\Kliko's modloader\Mod Profiles".
  2. Open the folder of the Mod Profile that you want to add your mod to, or create a new one.
  3. Run 'Modloader Menu' and make sure you have selected the right Mod Profile.

Mod Formatting

Mod Profiles should be made with the right file structure:

  • "%localappdata%\Kliko's modloader\Mod Profiles"
    • Mod Profile
      • Mod
        • content
        • ExtraContent
        • . . .
      • Mod
      • . . .
    • Mod Profile
    • . . .

Mod Priorities

  1. Go to "%localappdata%\Kliko's modloader\Program Files\config".
  2. Open mod_profiles.json.
  3. From here you can set up mod priorities, make sure to use the right format.
    • Add a comma "," after each Mod and Mod Profile except for the last one.

Mod Priority Formatting

Mod priorities should be entered with the following format:

{ "Mod Profile Name": { "1": "Mod Name", "2": "Mod Name" }, "Mod Profile Name": { "1": "Mod Name" } }

Adding FastFlags

Quick Guide

  1. Go to "%localappdata%\Kliko's modloader\Program Files\config".
  2. Open fastflag_profiles.json.
  3. From here you can edit your FastFlag profiles, make sure to follow the right format.
    • Add a comma "," after each FastFlag and FastFlag Profile except for the last one.
  4. Run 'Modloader Menu' and make sure you have selected the right FastFlag Profile.

Formatting

FastFlag Profiles should be entered with the following format:

{ "FastFlag Profile Name": { "FastFlag Name": "FastFlag Value", "FastFlag Name": "FastFlag Value" }, "FastFlag Profile Name": { "FastFlag Name": "FastFlag Value" } }

Selecting Mod / FastFlag Profiles

Instructions

  1. Open the Modloader Menu.
  2. Choose 'Change Mod Profile' or 'Configure FastFlags'.
  3. Select the mod profile or FastFlags profile that you want to use.
  4. Go back and choose 'Save & Exit'.
